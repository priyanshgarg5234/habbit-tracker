📌 Habit Tracker Application
Overview
The Habit Tracker is a simple yet powerful tool designed to help users build and maintain positive habits. It allows you to set goals, track progress, and visualize consistency over time.

Features
✅ Add, edit, and delete habits

📊 Track daily/weekly/monthly progress

🔔 Reminders and notifications to stay on track

📈 Analytics dashboard to view streaks and completion rates

Tech Stack
Frontend: JavaScript, React

Backend: Node.js/Express

Database: MongoDB
